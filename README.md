# Student Management System 🎓

A console-based application to manage student records built with Core Java.

## Features
- Add new students
- View all students
- Search student by name
- Update student details
- Delete student records
- Data persists using File I/O

## Technologies Used
- Java (Core)
- OOP — Classes, Objects, Encapsulation
- File Handling — FileWriter, Scanner

## How to Run
1. Clone the repo
2. Open in Eclipse as a Java Project
3. Run Main.java
